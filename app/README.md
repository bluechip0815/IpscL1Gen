# IPSC Level I Match-Unterlagen Generator

Browser-only React/Vite-App zum Erstellen von IPSC Level I Sanktionierungsunterlagen.

## Start

```bash
npm install
npm run dev
```

Die App laeuft standardmaessig unter `http://localhost:5173`.

## Build

```bash
npm run build
```

## Regelbasis

- GROI/IPSC.de: Matchsanktionierung und Anforderungen an Stage Briefings ab Januar 2026.
- IPSC Handgun Competition Rules, Januar 2026.

Vor Einreichung sollte der erzeugte ZIP-Inhalt gegen die jeweils aktuellen offiziellen Formulare und Hinweise geprueft werden.
